# line-pay-sdk-go
LINE Pay API SDK for Go
[LINE Pay Developers Website](https://pay.line.me/developers/apis/onlineApis)

# Supported LINE Pay v3 APIs
---------------
- ✅ Request API 
- ✅ Confirm API 
- ⛔️ Capture API
- ⛔️ Void API
- ⛔️ Refund API
- ✅ Payment Details API 
- ⛔️ Check Payment Status API
- ⛔️ Check RegKey API
- ⛔️ Pay Preapproved API
- ⛔️ Expire RegKey API
